# 📺 Vavuu Web Player

Live-TV Streams via Vavoo – Playlist-Generator & eingebetteter HLS-Player.

## Features

- 📥 **Playlist Generator** – M3U8-Datei für VLC, Kodi, TiviMate, IPTV Smarters
- 📺 **Live Player** – Streams direkt im Browser via hls.js
- 🌍 Alle verfügbaren Länder/Gruppen mit Sender-Anzahl
- 🔗 Optionale URL-Auflösung (direkte CDN-Links)
- `User-Agent: libmpv` in jeder Playlist-Zeile

## Nutzung

1. **Tab „Playlist Generator"**: Land wählen → Playlist generieren → Download
2. **Tab „Live Player"**: Land wählen → Sender laden → Sender wählen → Abspielen
